About Our Application

Companies

Our platform provides comprehensive management of company-related information. You can perform the following actions:

  • 📋 List companies - View all companies registered on the platform.
  • 🔍 Get a specific company by ID - Retrieve details of a company using its unique ID.
  • ➕ Create a company - Add a new company to the platform.
  • ✏️ Update a company - Modify the details of an existing company.
  • 🔄 Add/remove users to/from a company - Manage the users associated with a company.
  • 🗑️ Delete a company - Remove a company from the platform.

Users

Manage user accounts with ease. The following functionalities are available:

  • 📋 List users - View all users registered on the platform.
  • 🔍 Get a specific user by ID - Retrieve details of a user using their unique ID.
  • ➕ Create a user - Add a new user to the platform.
  • ✏️ Update a user - Modify the details of an existing user.
  • 🚫 Deactivate a user - Set a user’s status to inactive without deleting their information.
  • 🗑️ Delete a user - Remove a user from the platform.
